

Fishing Gym Docs
==============================

The `Fishing Gym <https://github.com/boettiger-lab/gym_fishing>`_ is a collection of `OpenAI-gym <https://github.com/openai/gym/>`_ reinforcement learning environments that examine classic fisheries management problems. 

Github Repository: `https://github.com/boettiger-lab/gym_fishing <https://github.com/boettiger-lab/gym_fishing>`_

Main Features
--------------

- Unified structure for all environments
- PEP8 compliant (unified code style)
- Documented functions and classes
- Tests, high code coverage and type hints
